A modern **Student Management System** built using **ReactJS**, **Vite**, **Axios**, and **JSON Server**. This application allows users to efficiently manage student records with complete **CRUD (Create, Read, Update, Delete)** functionality.

---

## 🚀 Live Demo

🌐 **Frontend:**  
https://student-management-system-react-fivnjogd1.vercel.app

🔗 **Backend API:**  
https://student-management-backend-yzrj.onrender.com/students

---

## ✨ Features

- ➕ Add New Student
- 📋 View All Students
- 👤 View Student Details
- ✏️ Update Student Information
- 🗑 Delete Student Record
- 🔄 REST API Integration using Axios
- 📱 Responsive User Interface
- ⚡ Fast Performance with Vite

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/jashwanth345/student-management-system-react.git
```

### Navigate to Project

```bash
cd student-management-system-react
```

### Install Dependencies

```bash
npm install
```

### Start Frontend

```bash
npm run dev
```

---

## 📡 Backend Setup

```bash
cd backend
npm install
npm start
```

JSON Server runs on:

```
http://localhost:5000/students
```
